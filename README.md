# 💸 Price Tiers Web Page

A responsive pricing panel webpage built using **HTML** and **CSS**, featuring three subscription tiers: **Personal**, **Small Team**, and **Enterprise**. This project demonstrates modern layout techniques using **Flexbox**, basic media queries for responsiveness, and clean UI design principles.

## 🚀 Features

- 💡 Clean and minimal UI
- 📱 Fully responsive layout (mobile and desktop)
- 🎨 Custom icons and pricing visuals
- 🎯 Styled with Google Fonts and custom CSS
- 🧩 Three-tier pricing cards with hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Google Fonts
